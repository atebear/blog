## **My backup site is now open source**

Remember when I mentioned my backup site a couple of posts ago?

Well...

It's now **open source**!

*"But wait... didn't you say you wanted to stay anonymous under your pen name, Ate Bear?"*

Yes, and I still do.

Originally, my backup lived in a private repository on my personal GitHub account, and there was no way I was making that public.

So instead, I created a brand new GitHub account under the username **atebear**.

From there, I created a new public repository, installed Hugo-BearBlog as a submodule, transferred everything from my private repository, and started backing up my BearBlog site there instead.

...

The last backup in my private repository was from May 31, so it was already outdated.

I transferred everything first without changing anything.

Then over the last two days, I updated everything to match the current version of my site.

While doing that, I also noticed Hugo-BearBlog was missing some features that BearBlog has, so I decided to implement them myself.

Features I added because BearBlog has them:

- Highlighted text using the "`<mark>`" HTML tag
- "Last edited" timestamps (like Last edited: 3 hours ago)
- Redirect aliases

Features I added that neither BearBlog nor Hugo-BearBlog has:

- Automatic page and post titles
- Custom footer text/citation
- Blog list titles
- Alerts and callouts

At this point...

I've customized almost every part of the theme. 🤭
