Another site update today...

I created some banners, cards, and logos for this blog with the help of [ChatGPT](https://chatgpt.com).

I don't think I'll use all of them, but somehow I've started treating [Ate Bear Blog](/) like it's a real brand. I'm just going with the flow at this point. 🤭

You can visit [my Collections page](/collections) if you'd like to see them.

I also spent some time exploring different websites today.

It's honestly mind-blowing how deep the rabbit hole goes once you start exploring the Indie Web.

Last month, I discovered [Neocities](https://neocities.org), and through it I was introduced to a bunch of cool and amazing websites. I didn't even know there was a name for this whole community until today.

Then there are:

[SlashPages](https://slashpages.net)

[Webmentions](https://webmention.net)

[Indie Web](https://indieweb.org)

[Webrings](https://xn--sr8hvo.ws)

[Nekoweb](https://nekoweb.org)

[Status.Cafe](https://status.cafe)

I'll leave links to all of them so you can explore them for yourself.

The ideas of owning your own content and decentralization really resonate with me.

What's funny is that I didn't realize I had already been doing a little bit of that since 2023 when I started making my own completely free websites.

I definitely want to support and participate in this movement.

For this blog, I'll probably implement some slash pages—but only the ones I actually need.

That said, I don't think I'll be talking much more about Indie Web things in future posts.

This will probably be the last post about it because what I really want to share here are my real-life stories, not my tech-nerd Indie Web adventures.

See you in the next one! 🐻
